Hotel Booking Cancellation Analysis (Excel Project)

Overview

This project is based on a real-world hotel booking dataset. The main objective is to analyze booking cancellations and understand customer booking patterns.
The project is created using Microsoft Excel with data cleaning, formatting, and analysis using Pivot Tables and dashboards.

Objectives
Analyze cancellation trends for different hotel types
Identify which month/year has the highest cancellations
Study guest type behavior (Single, Couple, Family)
Compare reserved room vs assigned room impact
Understand booking status distribution (Check-Out, Cancelled, No-Show)

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Data Cleaning & Formatting
Dashboard Creation

Conclusion

This Excel-based project provides useful insights into hotel booking behavior and cancellations. It helps hotel management understand trends and take action to reduce cancellations and improve customer experience.
